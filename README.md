# lyrics-worker

## Getting Started

### Environment Variables
以下の環境変数が必要です
```env
# Common
SITE_DOMAIN=

# last.fm
LASTFM_USER_ID=
LASTFM_API_KEY=

# Spotify
SPOTIFY_EMAIL=
SPOTIFY_PASSWORD=
SPOTIFY_CLIENT_ID=
SPOTIFY_CLIENT_SECRET=
SPOTIFY_REFRESH_TOKEN=

# Twitter
TWITTER_CK=
TWITTER_CS=
TWITTER_AT=
TWITTER_ATS=

# Supabase
SUPABSE_URL=
SUPABSE_ANON_KEY=
```

### Installation
自力で頑張ってください
